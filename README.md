# JTS Logistics Portal

Production-ready Node.js / Express logistics portal with Admin, Dispatcher, Driver and Broker roles.

## Local start

```bash
npm install
npm start
```

Open: http://localhost:4000

Initial local admin:

```text
Email: admin@jtslogistics.local
Password: admin123
```

Change this password immediately after first login.

## Render + GitHub deployment

1. Push this folder to a new GitHub repository.
2. In Render, create **New Web Service** from the GitHub repository.
3. Render will read `render.yaml` automatically.
4. Set these environment variables in Render:
   - `PUBLIC_APP_URL` = your Render URL
   - `JWT_SECRET` = generated by Render or your own long secret
   - `VAPID_PUBLIC_KEY` and `VAPID_PRIVATE_KEY` if push notifications are enabled
   - `VAPID_SUBJECT=mailto:admin@jtslogistics.net`
5. Deploy.

## VAPID keys

```bash
npx web-push generate-vapid-keys
```

Copy the generated public/private keys into Render environment variables.

## Roles

- Admin
- Dispatcher
- Driver
- Broker

## Security note

Do not commit real production secrets. Use Render environment variables for production.
