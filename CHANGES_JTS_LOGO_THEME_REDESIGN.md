# JTS Logo Theme Redesign

Implemented a visual-only redesign based on the JTS logo palette.

## What changed

- Added a new global override stylesheet: `public/assets/jts-logo-theme-redesign.css`.
- Injected the stylesheet into all HTML pages in `public/` as the final CSS layer.
- Updated the PWA/browser theme color from dark navy to JTS teal.
- Redesigned visual presentation for:
  - login/landing screen,
  - headers and navigation,
  - page hero sections,
  - cards and panels,
  - forms and inputs,
  - buttons,
  - tables,
  - floating/role navigation,
  - mobile and tablet layouts.

## Functionality

No backend routes, API calls, authentication logic, data files, or JavaScript business logic were changed.
The update is intended to preserve all existing functionality and only modify the interface appearance.

## Main colors used

- JTS teal: `#0BA5A2`
- Soft aqua: `#A1E7E5`
- Graphite: `#6A6C6D`
- Deep graphite: `#142326`
- White / light background: `#F7FBFB`
